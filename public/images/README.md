# Imágenes del sitio

- `branding/`: logo, isotipo y versiones de marca.
- `institution/`: edificio, autoridades, comunidad y actividades institucionales.
- `careers/`: portadas e imágenes de carreras.
- `news/`: imágenes destacadas de noticias.

Usar nombres en minúsculas, sin espacios ni tildes. Ejemplo: `logo-horizontal.svg`.
